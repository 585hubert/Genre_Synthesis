The notebook provides a simple git clone of the RCNN used.

Upon installation, replace the contents of the data folder with the content inside of this zipped folder: https://drive.google.com/file/d/1AvD77ReysHv7Gfu6hWm9RHOm1m6me3f-/view?usp=sharing

Inside of the main.py module, you can then specify which of the train and test csvs you wish to use.
